HTTP

  • URL的组成：

    • http://www.abc.com:8080/news/index.asp?boardID=5&ID=24618&page=1#name
      1. http -> HTTP协议
      2. www.abc.com -> 域名
      3. 8080 -> 端口号
      4. news -> 虚拟目录
      5. index.asp -> 文件名
      6. ?boardID=5&ID=24618&page=1 -> 查询字符串
      7. #name -> 哈希字符串
    • www.m.baidu.com
      1. .com -> 顶级/一级域名
      2. baidu.com -> 二级域名
      3. m.baidu.com -> 三级域名
      4. www.m.baidu.com -> 四级域名
